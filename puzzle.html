メイク10パズル（メイクテンパズル）：4つの数字で10を作る算数パズル

4つの数字と算数の記号「＋ / － / × / ÷ / ()」をうまくならべて10を作る算数パズル

  • メイク10パズル（メイクテンパズル）：標準問題
    • 「＋ / －」：小学1年生～
    • 「＋ / － / ×」：小学2年生～
    • 「＋ / － / × / ÷」：小学3年生～
    • 「＋ / － / × / ÷ / ()」：小学4年生～
  • メイク10パズル（メイクテンパズル）：チャレンジ問題
    • 標準問題をクリアできた人向けのメイク10パズル（メイクテンパズル）のチャレンジ問題

素因数分解チャレンジ

数字を制限時間内に簡単なかけ算に分解する「素因数分解」にチャレンジ。正解するほど問題が難しくなります

数列の推理パズル

数字のならび方の法則を発見して、□に入る数字を推理する問題です

  • 標準問題
    • 等差数列（とうさすうれつ）：「増える数」が同じ数列
    • 階差数列（かいさすうれつ）：「増える数」が等差数列になっている数列
    • フィボナッチ数列（フィボナッチすうれつ）：「増える数」が「1つ前の数」で決まる数列

    算数パズルが好きな人にオススメのドリル

    • フラッシュ暗算にチャレンジ
    • 先にまとめるかけ算の暗算ドリル
